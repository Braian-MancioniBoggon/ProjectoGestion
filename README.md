# Proyecto Gestion

## Meta:
Crear una aplicación web utilizando React que permita gestionar una lista de tareas con los siguientes campos:
- Id
- Ingreso
- Cliente
- Contacto
- Descripción
- Presupuesto
- Total
- Seña
- Saldo
- Estado
- Entrega
